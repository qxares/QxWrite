# QxWrite Bricks Documentation

## Overview
Modular components for QxWrite and beyond—stackable, reusable bricks.

### 1. InsertBrick
- **Purpose**: Inserts images into QTextEdit via QTextDocument.
- **Files**: `insertbrick.h`, `insertbrick.cpp`

### 2. SaveBrick
- **Purpose**: Saves QTextEdit content to a text file.
- **Files**: `savebrick.h`, `savebrick.cpp`

### 3. GeneralManagerBrick
- **Purpose**: Coordinates bricks, wires them to UI.
- **Files**: `generalmanagerbrick.h`, `generalmanagerbrick.cpp`

## Notes
- **Current Features**: Basic editor with save and image insertion.
- **Pending**: More bricks (Bold, Zoom, etc.), SDI transition.
- **Future**: Toolbar, format extensions, tables.
